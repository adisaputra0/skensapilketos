## Skensa Pilketos

Platform pemilihan Ketua OSIS di SMKN 1 Denpasar, yang dikenal sebagai skensapilketos, telah dikembangkan oleh sebuah tim yang terdiri dari dua individu. Dalam proyek ini, saya memfokuskan diri pada pengembangan sistem backend, sementara rekan saya memiliki tanggung jawab terhadap desain dan tampilan situs web.
